# Basic MongoDB Commands

### **Description:**
> ***get a list of commands.***
---------------------------------------

<strong>Command: </strong>

```MongoDB
db.help()
```
----------------------------------------

### **Description:**
> ***print a list of all databases on the server.***
---------------------------------------

<strong>Command: </strong>

```MongoDB
show dbs
```
----------------------------------------

### **Description:**
> ***create new database "myTestDB".***
---------------------------------------

<strong>Command: </strong>

```MongoDB
use myTestDB
```
